# Standalone Frontend for Car Rental Project

Push this folder to GitHub for a **live demo** of your frontend.

## Setup

1. Copy all car images from `../../static/cars_pics/` to `assets/images/`
2. All HTML files are ready to deploy as-is, no build step needed

## Deploy Options

### GitHub Pages (Free & Easy)
```bash
# 1. Create a new repo called "car-rental-frontend"
# 2. Push this folder to the repo
# 3. Go to Settings → Pages → Deploy from main branch in /frontend/live
# 4. Your site will be live at: https://yourusername.github.io/car-rental-frontend
```

### Netlify (Drag & Drop)
```bash
# 1. Zip this folder
# 2. Go to netlify.com
# 3. Drag and drop the zip file
# 4. Get a live URL instantly
```

### Vercel
```bash
# 1. Push this folder to GitHub
# 2. Go to vercel.com
# 3. Import your GitHub repo
# 4. Deploy with one click
```
